Here are some of the projects I've worked on in my free time, in alphabetic order.

- [Bookmarkleter](https://github.com/seguri/bookmarkleter): bookmarklet web editor
- [devmap](https://devmap.ch): a map of swiss software houses with direct links to careers page
- [Final Obsession](https://github.com/seguri/final-obsession): an intelliJ IDEA plugin that adds the `final` modifier to each possible local parameter and variable
- Job Hunt: manage job applications and display them in a RAV-compatible format for quick copy & paste
- Kinderchef: a meal planning tool used by a Kita to manage attendances, meals and allergies
- [Lock](https://github.com/seguri/Lock): a screen locker for Android
- Places: injest Google Maps takeouts and visualize your saved places
- [PowerToys-Run-JohnnyDecimal](https://github.com/seguri/PowerToys-Run-JohnnyDecimal): a plugin for PowerToysRun that allows you to quickly navigate to your Johnny Decimal folders
- [springpom](https://github.com/seguri/springpom): the parent pom of my Spring Boot projects
- Tastoni, an android app meant to be used by non-tech savvy people that I can manage remotely to show different kinds of big buttons to call a specific number, navigate to a specific position, open a specific phone setting, etc.
- Training Center: analytics for martial arts gym by collecting attendances, members & belts, techniques, and videos
- Waschplan: laundry scheduler, useful in countries where the wasching machines are shared among tenants
- [x-zm-signature](https://github.com/seguri/x-zm-signature): a library for verifying Zoom webhook events through the `x-zm-signature` and `x-zm-request-timestamp` headers
- several discord and telegram bots

Some of the technologies used in these projects:

- Java, Kotlin, Spring Boot, Hibernate, Postgres, Redis
- Python, Django, Pandas, Plotly, Kaleido, discord.py
- JavaScript, TypeScript, Node.js, Cloudflare Workers, Hono, Biome
- C#
- React, TanStack Query, Tailwind CSS
- Prometheus, Grafana
